# Intelligent Analysis of Biomedical Images – Practical Homeworks (Fall 2025)

This repository contains my solutions to the **practical homework assignments** for the *Intelligent Analysis of Biomedical Images* course.  
Each homework explores a different aspect of biomedical image analysis – from classical image processing to deep learning and graph neural networks.

## 📚 Homework Overview

| Folder | Topics / Techniques |
|--------|----------------------|
| **HW1 – Classic Image Processing** | Filters, edge detection, morphological operations, segmentation (e.g., Otsu, watershed) |
| **HW2 – CT & MedMNIST** | CT image analysis, classification on MedMNIST dataset (e.g., pneumonia detection) |
| **HW3 – UNet & CellProfiler** | U-Net architecture for segmentation, CellProfiler for high‑throughput cell profiling |
| **HW4 – GNN & Voxelmorph** | Graph Neural Networks (GNNs) for medical images, VoxelMorph for deformable registration |
